# Roles de widget
Definen patrones comunes de interactividad. Suelen requerir de *Javascript*.

## Lista de referencia
### Simples
#### Con equivalencia en HTML5
- `button`
- `checkbox`
- `gridcell`
- `link`
- `menuitem`
- `menuitemcheckbox`
- `menuitemradio`
- `option`
- `progressbar`
- `radio`
- `search`
- `textbox`

#### Sin equivalencia en HTML5
- `scrollbar`
- `separator` (cuando recibe el foco)
- `slider`
- `spinbutton`
- `switch`
- `tab`
- `tabpanel`
- `treeitem`

### Composite (Compuestos-contenedores)
#### Con equivalencia en HTML5
- `listbox`
- `radiogroup`

#### Sin equivalencia en HTML5
- `combobox`
- `menu`
- `menubar`
- `tablist`
- `tree`
- `treegrid`
- `grid`
