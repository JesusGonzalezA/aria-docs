# Nombre accesible

El nombre accesible de un elemento es el resultado de un cálculo bastante complejo que intentaremos simplificar. Puedes encontrar toda la información acerca del mismo en [la documentación de w3, accname/computation-steps](https://w3c.github.io/accname/#computation-steps).

## Algoritmo simplificado

El nombre accesible de un elemento será el primero que haya definido, según este orden:

1. **aria-labelledby**
2. **aria-label**
3. **HTML label, alt ó contenido dentro de la marca**
4. **title**: no es recomendable usar *title* como método para nombrar un elemento. Se usa para indicar información extra, no esencial, ya que no es accesible para todos los usuarios.

### Ejemplos

#### Ejemplo nombre: mecanismos HTML
```jsx
<label>
    Nombre
    <input type="text" name="fname" />
</label>

<button>Guardar</button>
```

### Ejemplo nombre: aria-label
```jsx
<label for="phone__prefix">Teléfono fijo</label>

<input id="phone__prefix" aria-label="Prefijo del teléfono" size={3} maxLength={3} type="text" style={{ padding: "0.5rem" }} />
 - 
<input id="phone__rest" aria-label="Número de teléfono sin prefijo" size={6} maxLength={6} type="text" style={{ padding: "0.5rem" }} />
```

<div style={{ background: "white", color: "black", padding: "3rem", border: "solid white 1px", borderRadius: "0.5rem", border: "solid 1px black"}}>
    <label for="phone">Teléfono fijo</label>

    <input id="phone" aria-label="Prefijo del teléfono" size={3} maxLength={3} type="text" style={{ padding: "0.5rem" }} /> - <input id="phone2" aria-label="Número de teléfono sin prefijo" size={6} maxLength={6} type="text" style={{ padding: "0.5rem" }} />
</div>
