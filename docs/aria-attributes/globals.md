# Atributos globales
Aplican a todos los roles de *ARIA*, a no ser que esté específicamente deshabilitado para un rol concreto. Por ejemplo, el atributo <code>aria-label</code> para el rol <code>presentation</code>.

- `aria-controls`
- `aria-current`
- `aria-describedby`
- `aria-description`
